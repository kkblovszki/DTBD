how does it work in-\/code\+: When creating a new simulator, the following steps need to be taken\+:
\begin{DoxyItemize}
\item Run constructor\+:
\item load the APIFunction\+Pass\+Through from the NS3\+\_\+config.\+yaml file
\item Return a \mbox{\hyperlink{classNS3__mockup__interface}{NS3\+\_\+mockup\+\_\+interface}} object from the create\+Simulator function
\end{DoxyItemize}

When the Load\+Configuration function is called, the following steps need to be taken\+:
\begin{DoxyItemize}
\item load the persistent data from the NS3\+\_\+config.\+yaml file
\begin{DoxyItemize}
\item Get the supported parameters
\end{DoxyItemize}
\item Crosscheck the supported parameters from the simulator with the inputted parameters
\item If the parameters are supported, then pass them to the simulator else throw an error saying that the specific parameter(s) are not supported
\item It then creates a list of the supported metrics and parameters from the simulator
\end{DoxyItemize}

When the Load\+Metrics function is called, the following steps need to be taken\+:
\begin{DoxyItemize}
\item Convert the standardized metrics to the simulator specific metrics types
\end{DoxyItemize}

When the Load\+Parameters function is called, the following steps need to be taken\+:
\begin{DoxyItemize}
\item Get the inputted parameters from the benchmark yaml file
\item Convert the standardized parameters to the simulator specific parameters types i.\+e. \mbox{\hyperlink{structBuildOptions}{Build\+Options}} Type
\end{DoxyItemize}

When the Parse\+To\+NS3\+Command\+Line function is called, the following steps need to be taken\+:
\begin{DoxyItemize}
\item Parse the inputted parameters to an NS3 command line string for building the simulation ~\newline

\end{DoxyItemize}

When Run\+Simulation is called, the following steps need to be taken\+:
\begin{DoxyItemize}
\item Run the parsing function to parse the inputted parameters to an NS3 command line string for building the simulation
\item Set up a filestream to read the output from the simulation
\item Run the simulation
\item Read the output from the simulation
