\doxysubsection{Detailed Description}
Implements parser functions for the YAML configuration files \mbox{\hyperlink{classParser_af734dbfe88530641ae650f5ae0be0b88}{parse\+Scenario}}, \mbox{\hyperlink{classParser_a70dd387bb9f514e4a4b52da82123639e}{parse\+Benchmark}} @parse\+Simulation\+Strategy. 

Must follow the Main Rules of the YAML configuration file (which is so far)\+:
\begin{DoxyEnumerate}
\item If no benchmark exists no Simulator can be determined thus no scenario can be created.
\item A single benchmark must consist of atleast 1 simulator with 1 scenario connected.
\item The scenario must have atleast a name defined 
\end{DoxyEnumerate}
