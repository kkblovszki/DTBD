Detailed Description

SimulatorInfo structure that holds the information about a simulator.

Parameters
simulatorName- the name of the simulator
simulatorVersion- the version of the simulator
nativeOutputType- the native output type of the simulator
functions- the functions that the simulator supports
parameters- the parameters that the simulator supports
metrics- the metrics that the simulator supports
buildOptions- the build options that the simulator supports

This structure is used to hold the information about the simulator that is read from the simulation configuration file. The entirety of this structure isn't optional, this is since the framework needs to know what simulator it is working with, what version it is and what it supports. Therefore a new simulator must be defined with a simulation configuration file.
