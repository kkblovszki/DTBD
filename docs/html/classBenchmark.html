◆ LoadSimulationConfig()

void Benchmark::LoadSimulationConfig ( std::string  configPath)

Using the library yaml-cpp, load the simulation configuration file and store the data in the Benchmark object convert the config into an accessable ordered map, to get the individual simulations names, get the simulator type, and the listener type Runs the parser parseScenario, parseSimulator, and parseListener to parse the simulation configuration file into a predetermined format, with a ruleset.

Loads the simulation configuration file.

Parameters
configPath

The simulation configuration file is a YAML file that contains the simulation scenarios and their respective simulation strategies. The simulation configuration file is loaded into a YAML::Node object and then parsed using the @Parser class.

Parameters
configPath- the path to the simulation configuration file
Exceptions
std::runtime_errorif the simulation configuration file is not found
std::runtime_errorif the simulation configuration file is empty
