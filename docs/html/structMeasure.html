Detailed Description

Measure structure that holds the information about a measurement.

Parameters
label- the label of the measure
readings- the readings of the measure it consist of a map of metrics and a vector of strings pr. metric

This structure is used to hold the information about the measure that is read from the simulation configuration file. The Measure structure itself contains a map of multiple Metrics. The Metrics structure contains a name and a unit.
